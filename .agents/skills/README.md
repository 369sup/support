# Repository skills

Create repository-scoped skills here as:

```text
.agents/skills/<skill-name>/
├── SKILL.md
├── agents/openai.yaml    # optional
├── scripts/              # optional
├── references/           # optional
└── assets/               # optional
```

Do not add an empty example skill: a vague `description` can trigger it for the
wrong tasks. Follow [../AGENTS.md](../AGENTS.md) when adding a real workflow.
